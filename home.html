stuff i did or am trying to do

media server

set up a jellyfin server that i'm running off of a mac mini that i've converted into a NAS for my stuff. was using tailscale in order to access it remotely/share with some family but was having to do some trickery to get them all on the tailscale network. ended up using cloudflare tunnels to expose it so I can just access it normally. added google auth as well so it's not just exposed which was nice.


link sharing

other than hosting the media server figured out i was often sharing links to files that i was mainly uploading to google drive and then sending that link. it was turning into a decent hassle so i found a service called pingvin-share that lets me upload files and share links to them without using google drive. also got it running through the cloudflare tunnel which makes sharing with other people a lot easier.
